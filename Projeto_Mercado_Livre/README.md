# Teste Funcional — Mercado Livre

## Objetivo

Avaliar o funcionamento de funcionalidades essenciais do site Mercado Livre, com foco no fluxo de busca de produtos, aplicação de filtros, visualização de página de produto e manipulação do carrinho de compras.

## Escopo do Teste

Os seguintes cenários foram avaliados:

- Busca de produto
- Aplicação de filtros
- Acesso à página de produto
- Adição de produto ao carrinho
- Remoção de produto do carrinho

## Ambiente de Teste

- Sistema Operacional: Windows 10 Pro
- Navegador: Google Chrome
- Tipo de teste: Manual funcional
- Data do teste: 06/03/2026
