# Teste Exploratório – Amazon Brasil

## Objetivo

Identificar e documentar possíveis falhas na funcionalidade de busca e filtragem de produtos no site Amazon Brasil, avaliando o comportamento do sistema durante a navegação e aplicação de filtros.

---

## Escopo do Teste

O teste exploratório foi focado nas seguintes funcionalidades:

- Pesquisa de produtos
- Aplicação de filtros de preço
- Ordenação de resultados
- Navegação entre páginas de produtos

---

## Ambiente de Teste

Sistema testado: Amazon Brasil  
Dispositivo: Desktop / Notebook  
Sistema Operacional: Windows 10 Pro  
Navegador: Google Chrome  
Tipo de teste: Teste manual exploratório

---

## Bugs Identificados

Durante a execução do teste exploratório foram identificados os seguintes problemas:

### BUG-AMZ-001
Filtro de preço retorna produtos fora da faixa selecionada.

### BUG-AMZ-002
Ordenação por menor preço ignora filtro de preço aplicado, exibindo produtos fora da faixa definida.
